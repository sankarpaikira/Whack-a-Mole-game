## 🚀 Phase 1: Month 1 — Basics & Frameworks

### 📚 Week 1: Core Programming Basics
*Daily Habit: 1 Hour of basic syntax on HackerRank / Exercism.*
- [ ] Learn/Review deep copies vs. shallow copies in Python & JS
- [ ] Practice Python List Comprehensions and Dictionary manipulations
- [ ] Implement OOP concepts: Classes, Inheritance, and Method Overriding in Python
- [ ] Master JavaScript ES6+ features: Arrow functions, Array methods (`.map()`, `.filter()`, `.reduce()`)
- [ ] Understand JavaScript Event Loop, Promises, and `async/await` syntax

### 🌐 Week 2: Web Fundamentals & Version Control
*Daily Habit: Build a small landing page and push changes daily.*
- [ ] Build a responsive layout using raw HTML5 and CSS Flexbox/Grid
- [ ] Learn CSS Variables and handling media queries for mobile-responsive design
- [ ] Learn Git commands: `git checkout -b`, `git merge`, `git rebase`
- [ ] Configure Git to use [Conventional Commits](https://conventionalcommits.org) format
- [ ] Create a GitHub profile and host your first public repository

### 🐍 Week 3: Backend Frameworks (FastAPI)
*Daily Habit: 1-2 LeetCode Arrays/Strings/Hash Maps problems (Python).*
- [ ] Install FastAPI and Uvicorn; build a "Hello World" endpoint
- [ ] Understand HTTP methods (`GET`, `POST`, `PUT`, `DELETE`) and matching status codes
- [ ] Implement query parameters, path parameters, and request body validation using Pydantic
- [ ] Learn to handle CORS (Cross-Origin Resource Sharing) middleware configurations
- [ ] Build an in-memory CRUD application (no database yet) and test it on `/docs` Swagger UI

### 🎨 Week 4: Modern Frontend (React + TypeScript)
*Daily Habit: Practice TypeScript basic coding problems on BigFrontend.dev.*
- [ ] Initialize a React project using Vite with TypeScript template configuration
- [ ] Master React core features: Functional Components, Props, and `useState` hook
- [ ] Use `useEffect` hook to fetch data from your Week 3 FastAPI backend via `fetch`/`axios`
- [ ] Implement strict interface definitions for all API response states in TypeScript
- [ ] Build a dynamic, multi-component task view page with modern CSS/Tailwind

---

## 🐳 Phase 2: Month 2 — System Integration & DevOps

### 🐳 Week 5: Databases & Docker Containers
*Daily Habit: LeetCode Mediums focusing on Two Pointers / Sliding Window.*
- [ ] Install SQLAlchemy ORM and configure a local SQLite or PostgreSQL instance
- [ ] Create relational database models (e.g., User has many Tasks)
- [ ] Write a production-ready `Dockerfile` for the Python FastAPI backend
- [ ] Write a separate development `Dockerfile` for the React frontend application
- [ ] Combine them: Write a `docker-compose.yml` to spin up frontend, backend, and db in one network

### 🧪 Week 6: Automated Testing & Clean Code
*Daily Habit: LeetCode problems on Recursion, Linked Lists, and Stacks.*
- [ ] Set up **PyTest** and write automated unit tests for all FastAPI endpoints
- [ ] Set up **Vitest** or **Jest** to run component and rendering tests on React components
- [ ] Aim for and measure **>80% test coverage** on both applications
- [ ] Refactor codebase to abstract logic away from endpoints into dedicated service files
- [ ] Review Git commit history to ensure it looks clean, structured, and corporate-ready

### 🏗️ Weeks 7 & 8: The HENNGE Capstone Simulation
*Goal: Build a fully containerized, professional Jira/Trello Clone from scratch.*
- [ ] **Day 1-3:** Design database schemas and map out the frontend UI layout
- [ ] **Day 4-6:** Build the backend with production-grade JWT Token Authentication
- [ ] **Day 7-9:** Build the UI view layout in React+TypeScript, complete with dashboard forms
- [ ] **Day 10-12:** Dockerize the entire project architecture and write comprehensive unit tests
- [ ] **Day 13-14:** Create a pristine GitHub `README.md` containing operational setup scripts
